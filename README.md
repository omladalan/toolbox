## Permissões Necessárias

Para que o software funcione corretamente, é necessário garantir que o arquivo `toolbox.sh` tenha permissão de execução. Você pode definir isso utilizando o seguinte comando:

```bash
chmod +x toolbox.sh
```

Além disso, o atalho `toolbox.desktop` também deve ter permissão de execução. Execute o mesmo comando para garantir essa permissão.

## Observações sobre o Caminho do Ícone

Assumindo que o projeto esteja localizado na raiz da pasta pessoal, o software deve executar sem problemas. Vale ressaltar que o caminho do ícone não funcionará se você utilizar:

```
~/toolbox/arame.png
```

Certifique-se de ajustar o caminho do ícone conforme necessário para que ele seja reconhecido corretamente.
